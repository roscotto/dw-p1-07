Tenemos una especie de "calculadora", con 7 botones:

  • Tenemos 2 botones para ingresar valores numéricos.
  • Tenemos 4 botones para realizar una operación aritmética.
  • Tenemos un botón que muestra el resultado.

Ya tienen una función asignada a su onclick:

  • obtenerNumero1
  • obtenerNumero2
  • operacion
  • mostrar

El funcionamiento debe ser el siguiente:

  • Ingresar dos valores.
  • Seleccionar una operación.
  • Mostrar el resultado.

Se debe verificar que los datos sean válidos (en caso contrario notificarlo).

Se deben utilizar variables globales.

Extra: crear una función para ingresar y validar un número.
